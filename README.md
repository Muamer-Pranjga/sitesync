# SiteSync — Frontend

React frontend for the SiteSync job site management system. Built with Vite and connected to the SiteSync FastAPI backend.

## Live App
https://sitesync-app.netlify.app

## Backend API
https://sitesync-wrjy.onrender.com/docs

## Tech Stack
- React
- Vite
- Axios

## Features
- User registration and login
- View and create job sites
- Add and manage tasks per job site
- Update task status (Not Started / In Progress / Completed)
- Industrial construction-themed UI

## Getting Started
```
npm install
npm run dev
```

Visit `http://localhost:5173`
